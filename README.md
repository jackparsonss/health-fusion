## Info
- A basic health app for managing medications
- Schedule medication
- Look at a daily agenda of your medication
- Look for nearby locations

## Run
- `npm install`
- `sudo npm install -g expo-cli`
- `expo start`

## Tech
- React Native
- Firebase(authentication)
- Firestore(database)
- Google Maps API
- Google Autocomplete
